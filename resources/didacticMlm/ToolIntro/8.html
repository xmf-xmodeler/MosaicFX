8. Adding Values to RatingEnum

We must change the data type of our attribute "rating" from String to RatingEnum.


3. Step: Select Enumeration As The Data Type

If an attribute with a defined data type already exists, as in our case, right-click on the class with the respective attribute, select 'Attribute' and then 'Change type'. In the window that will open, select the attribute from the 'Select Attribute' drop-down menu and enter the name of your enumeration as the new data type. Hint: Close the window by clicking on "OK". In this case, Enter will not work.

If the attribute does not yet exist, first add an attribute (as explained earlier) and enter the name of your enumeration as the data type. Hint: You can also select your enumeration from the drop-down menu under 'Type'. To display it, you need to first check the box next to 'Show all data types'.
