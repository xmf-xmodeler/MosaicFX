5. Exemplary Objects Creation

Let us create exemplary objects now. First, we add the movie “Joker” which has a duration of 122 minutes. Note that the name of a software object should not be confused with the name of the domain object it represents. Object names are unique and essential for the communication between software objects. Domain names for objects may not be unique; there may be multiple movies that share the same title. We leave the object name as it is ("movie1") and enter “Joker” as the respective title of the object.

Let’s say Joker is part of a showing on 7 Nov 2024. We first instantiate a MovieShowing object (again: we do not change the object name) and change the date value.

Now we create a link between the object for the movie Joker and object for the movie showing of Joker.


How To Create Objects

An object must always be an instance of a class. To instantiate an class, you can either right-click on a class in the canvas and select 'Add instance' or click on class in the palette first and then anywhere on the canvas. In both cases, a window will open where you can specify your object. Note: This window does not allow individual slots to be defined.


How To Enter A Slot Value

To enter a value for an object attribute, reffered to as a slot value, you can either double-click on the green-highlighted area of the slot value or right-click on the object and select 'Change Slot Value'. In both cases, a window will open where you can input the value.


How To Create Links

To add a link, there are two ways:

1. You can right-click on one an object and select 'Link' and then 'Add link'. A line will appear which you can drag to the other appropriate object and click.

2. Alternatively, you can click on 'Link' in the left pane and then click one of the appropriate objects. Again, drag the arrow to the other object and click.

In both cases, the link becomes immediately visible.

Note: The creation of a link will fail if no corresponding association between classes has been specified
