4. Association between Movie and MovieShowing

We must now add an association between the class "Movie" and the class "MovieShowing" so that instantiated objects can be linked to one another. We name the association “shown_in”. Hint: the black arrow next to the association name indicates the intended reading direction. In this example, we want to express that a particular movie is shown in a particular movie showing.

We must still add the multiplicities on each end of the association. One movie (object) can be shown in multiple movie showings. It might also appear in none. We therefore enter a multiplicity of 0…*. One movie showing, on the other hand, must always show exactly one movie. We therefore enter a multiplicity of 1..1.


How To Add An Association

To add an association, there are two ways:

1. You can right-click on one a class and select 'Association' and then 'Add'. A line will appear which you can drag to the class you want to connect your association to. You confirm your selection by clicking on the class.

2. Alternatively, you can click on 'Association' in the palette and then click one a class. Again, drag the arrow to the other class and click.

In both cases, a window will open where you can define your association. Note: It is also possible to associate a class with itself.
